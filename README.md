# Akash Verma - AI Engineer Portfolio

A modern, interactive 3D portfolio website built with React, Three.js, and Vite.

## Features

- **3D Physics Simulations**: Interactive ballpit and text effects using `react-three-fiber` and `cannon`.
- **Glassmorphism UI**: Sleek, modern interface with glass-like elements.
- **Dynamic Profile Card**: 3D tilt-enabled profile card with holographic effects.
- **Floating Dock**: macOS-style dock for quick access to social links and resume.
- **Responsive Design**: Fully optimized for mobile and desktop experiences.

## Tech Stack

- **Frontend**: React 19, Vite
- **3D & Animation**: Three.js, React Three Fiber, React Three Cannon
- **Styling**: CSS Modules, Glassmorphism
- **Icons**: React Icons

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/heyakash876/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
